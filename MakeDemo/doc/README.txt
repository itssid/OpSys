1. Build Instructions:
	Go inside make directory and type make
2. For Executing the Build:
	Go inside bin directory and type ./newfinal
	final is name of exe created using script 1_compile.sh
3. Debugging instructions:
	gdb newfinal
4. C Source is available in src folder while headers are in include folder
5. For removing object files from obj directory and removing exe from bin folder run make clean inside make folder
